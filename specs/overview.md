# Project Overview: Evolution of Todo

## Summary
A task management application that evolves from a basic web interface to an AI-powered, cloud-native chatbot system.

## Current Goal
Upgrade the existing Streamlit "Bonsai" project to a full-stack Next.js and FastAPI application (Phase II of the Hackathon).

## Feature Map
- [ ] **Multi-User Auth**: Secure sign-in/up via Better Auth.
- [ ] **Persistent Tasks**: CRUD operations stored in Neon DB.
- [ ] **Filtering & Priority**: Search, tags, and priority levels.
- [ ] **AI Chatbot (Phase III)**: Natural language task management.

## Components
- `/frontend`: Next.js web application.
- `/backend`: FastAPI REST server.
- `/specs`: Specification-driven documentation.
